# Git 更新rep

## 拉取rep
>git pull

## 提交rep
>cd ~/Documents/dwbl417.github.io  
>git add . 
>git status 
>git commit -m "git" 
>git push

## git push免输密码
>cd ~
>touch .git-credentials
>vim .git-credentials
>添加https://用户名:*your token*@github.com
>git config --global credential.helper store

打开~/.gitconfig文件，会发现多了一项:
[credential]
helper = store
